# Customer Segmentation Analysis Using Python, SQL and Tableau

## 📊 Overview
This project focuses on analyzing customer data to understand different customer segments based on their purchasing behavior.

The dataset was cleaned and prepared using Python (Pandas). SQL was used to perform analysis and extract insights, and Tableau was used to visualize the results.

## 🎯 Objective
The main goal of this project is to group customers into different segments and understand their behavior for better business decision-making.

## 🛠️ Tools Used
- Python (Pandas, NumPy)
- SQL
- Tableau

## 🧹 Data Cleaning (Pandas)
- Handled missing values  
- Cleaned and standardized data  
- Converted data types  
- Removed duplicates  
- Created new columns for analysis  

## 📈 Analysis (SQL)
- Customer segmentation based on purchase behavior  
- Average spending per customer  
- Category-wise sales analysis  
- Customer frequency and engagement  
- Revenue contribution by different segments  

## 📊 Visualization (Tableau)
- Sales by category  
- Customer distribution  
- Segment-wise performance  
- Revenue trends  

## 🔍 Key Insights
- Identified high-value and low-value customers  
- Found customers with frequent purchases  
- Certain categories generate more revenue  
- A small group of customers contributes to a large portion of sales
